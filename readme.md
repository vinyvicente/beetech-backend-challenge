# Beetech challenge backend

## Simple Product Management

### Using
    * Docker
    * Node.JS 12
    * PostGRESQL
    * API uses Nest Framework, and front Express

### Build

```shell script
docker-compose up --build
```

#### Running Tests

To run tests, run ```docker exec -it beetech_backend npm run test```, like below:
